Install Theme

To install the theme:

  1. Please log in to your Shopify store, click Online Store.
  2. From there, click Upload theme.
  3. The Upload Theme dialog displays. Click Choose File to select the .zip file you want to upload, then click Upload.
  4. If your store is currently active and is using a different theme, you can select Publish theme button to publish what you have just uploaded to be an active theme.

Updating theme without losing old settings

Remember: Our instruction below is only applied for updating theme source losing settings in Theme Editor. All of customizations you made in js or code files will not be updated and you need to re-do

If your store is using Bagratica 1.0.0 version and you would like to update to newest theme source to have all of new features or new layouts. Please follow below instruction:

Firstly, please backup your current theme source. You should duplicate your current theme

Then, go to Edit Code -> open this file settings_data.json and COPY ALL

After that, please go to Newest theme source which you just installed into your site. Go to Edit Code -> open this file settings_data.json , delete and PASTE code you just copied into this file

If you copy settings in setting_data.json file in your old theme source, please search for "enable_multilang" in file and change "true" to "false" to be able to user new Shopify Languages function

Multiple Currencies and Automatic Currencies

Bagratica supports Shopify multiple currencies as well as built-in currency features. Please follow our instruction below to choose and setup

01: Shopify multiple currencies:

This feature is only available for stores using Shopify Payments for a list of supported countries. Please follow instruction of Shopify to enable Shopify Payments.

  • How to enable Shopify Payments

    • After you already enable Shopify payments correctly, go to Theme settings -> Money Option to choose this option:

    02: Multiple currencies:

    This feature is built in feature of Bagratica which allows you to show multiple currencies on your site (Note: This option will not have automatic currency feature by location)

    Go to Theme settings -> Money Option and choose: Enable Multiple Currencies?

    03: Automatic currencies by GEO location:

    This feature is built in feature of Bagratica which allows your customers to see the item price on their own currency. When you enable this feature, currency selector on your site will be disabled, customers will see price in their currency by location. (Note: At checkout page, currency will be the default currency of your store)

    Go to Theme settings -> Money Option and choose: Enable Automatic Currency?

    04: Automatic currencies by GEO location for a group of currencies you want:

    This feature is the combination of #2 and #3 above. It will help you allow your customers to see the product price in their currency within a group of currencies you want and setup in theme setting.

    Go to Theme settings -> Money Option and choose both of these options: Enable Multiple Currencies? and Enable Automatic Currency?

Create multi-level drop-down menu

To add a drop-down menu from the main menu of your online store:

  1. From your Shopify admin, go to Online Store > Navigation.

  2. Click the title of your main menu.

  3. Choose one of the menu items to be the header for your drop-down menu, or add a new menu item to be the header. If you don't want the header to link to anything, then you can enter # in the Link field when you add the menu item.

  4. Add menu items to include in the new drop-down menu. Click Add menu item, enter the name of the menu item, and enter or select a destination for the link.

  5. Click and drag the menu items to nest below the header item:

  6. Click Save menu.

How to show Mega Menu

Step 1: To show mega menu layout, you need to create multi-level drop-down menus first.

Step 2: In Themes --> Customize -> Sections -> Navigation.

Step 3: At Navigation you need to check on Display Mega Menu?.

Step 4: At Content click add content to choose style.

Step 5: Put items you'd like to show as Mega Menu at Mega Menu Style 1, Mega Menu Style 2, Mega Menu Style 3, Mega Menu Style 4 and Mega Menu Style 5 and choose the Style Which is suitable for your option in Settings.

Step 6: On your site, it will have 5 styles of Dropdown Mega Menu.

Mega Menu Style 1

You'd like to display SS2017 as Mega Menu Style 1, you need to:

  1. Create the SS2017 menu in Navigation and the multi-level drop-down menu belongs to SS2017.
  2. In settings of Themes --> Customize -> Sections -> Navigation, at "Content" click Style 1 and put the name ofSS2017into Mega Menu Item 1.

Mega Menu Style 2

You'd like to display Shop as Mega Menu Style 2, you need to:

  1. Create the Shop menu in Navigation and the multi-level drop-down menu belongs to Shop.
  2. In settings of Themes --> Customize -> Sections -> Navigation, at "Content" click Style 2 and put the name ofShop into Mega Menu Item 2.

Mega Menu Style 3

You'd like to display Lookbook as Mega Menu Style 3, you need to:

  1. Create the Lookbook menu in Navigation and the multi-level drop-down menu belongs to Lookbook.
  2. In settings of Themes --> Customize -> Sections -> Navigation, at "Content" click Style 3 and put the name of Lookbook into Mega Menu Item 3.

Mega Menu Style 4

You'd like to display Demo as Mega Menu Style 4, you need to:

  1. Create the Collections menu in Navigation and the multi-level drop-down menu belongs to Collections.
  2. In settings of Themes → Customize → Sections → Navigation, at "Content" click Style 4 and put the name of Collections into Mega Menu Item 4.

  3. All sub menus of this menu should link to collections. To show collection images, you need set up Collection Image in Admin → Products → Collections

    Size Chart

    You have 4 options to show Size Chart: Using a Product Type, Product Vendor, Different for each product or Use the same size chart for all products.


    1 - Product Type

    Upload image with the same name as the product-type-sizechart (please use JPG image). You need to go to Settings --> Files upload images here.
    Naming convention: lower cap and use "-" to replace white space.
    For example: with the product type "Loremous Comodous" the image name will be loremous-comodous-sizechart.jpg


    2 - Product Vendor

    Upload image with the same name as the product-vendor-sizechart (please use JPG image). You need to go to Settings --> Files upload images here.
    Naming convention: lower cap and use "-" to replace white space.
    For example: with the vendor "Benjamin Button" the icon name will be benjamin-button-sizechart.jpg


    3 - Use the same size chart for all products.


    4 - Different for each product

    Using of Variant images and Color Variant

    To create product with color variant as in our demo page (https://bagratica-demo.myshopify.com/products/minos-lementum-risus-cras-damos-dincidunts-comodous?variant=2244122214427), you need to follow these steps:


    A. Add images to your product variants

    1. From your Shopify admin, click Products, then click on the product you would like to add color variants.

    2. In Variants sections, click the image icon to assign an image to that variant.

    3. Select a product image or click Add image to upload a new image.

    4. Click Save .



    B. Add color variant icons

    After that, to show color swatch icons, you can use one of two below methods:

    1. Upload icon with the same name as the variant color name (please use PNG icon). You need to go to Settings --> Files upload images here.
    Naming convention: Lower cap and use "-" to replace white space.
    For example: with the option name "Modern Style" the icon name will be modern-style.png.
    Here is Youtube video link to help you understand more about how to show Variant images with our theme in your site.

    2. Use color name for the option.
    Naming convention: No space and the name need to be existed in this table: http://www.w3schools.com/tags/ref_colornames.asp
    For example: MediumPurple or MistyRose, Black.

    Multiple Languages

    Bagratica offers to show 6 different languages on store. This is based on Shopify

    1. From your Shopify admin, click Online Store -> Click Customize for the theme you want to customize.
    2. At Theme Settings Tab -> Click Multi Languages, check Enable Multi Languages, then fill in the name of languages and upload icons you'd like to show.

    3. After that, go back to Settings -> Store languages,

      We recommend you to use this free app: Translation Lab, because it was tested and fully compatible with our theme

      Get App - Translation Lab

      We tested our theme very carefully with this app, all of things on theme can be translated into multi languages well. If you see there are blocks which have not been translated, it will not be bug theme, you must review your translation in app to make sure you already translated and translated correctly.


      Limitations of selling in multiple languages
      - The search function works only in the shop's primary language.
      - This feature doesn't support different domains per language.
      - Only compatible third-party translation apps are supported by this feature.
      - Tags (such as product tags, article tags, and blog tags) can't be translated.


      How to translate for Product Tags?

      As mentioned above that Shopify doesn't allow to translate product tags, so Product Filter on left sidebar will be affected if you want to use Multi Languages for it. In this case, you can use this manual way

      Go to each product and enter the text of product tags in the language you want for Filter options as image below

      Example: You have a Filter option: Color. In English it has "Yellow" color, you add tag "yellow" for your product, and in French language, you will need to add tag "jaune" for that product also
